# SideStore Repository Example

Пример репозитория приложения для SideStore (форка AltStore).

## Структура

- `app.json` - Метаданные приложения
- `releases/` - Директория с релизами приложения

## Использование

1. Добавьте URL этого репозитория в SideStore
2. Приложение появится в списке доступных для установки

## Требования

- iOS 14.0+
- SideStore или AltStore
