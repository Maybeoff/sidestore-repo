## Инструкции по добавлению релиза

1. Соберите приложение в Xcode (Product → Archive)
2. Экспортируйте как .ipa файл
3. Загрузите на GitHub Releases
4. Обновите `app.json` с новым URL и версией
5. Обновите `apps.json` если нужно
